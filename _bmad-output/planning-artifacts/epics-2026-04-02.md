## Epic 42: Fix Failing E2E Tests

**Goal:** The e2e tests written during previous sprints were never run against the implementation.
Running them now reveals that some features were not fully implemented. Fix all failing tests by
correcting the implementation code — do not modify the tests themselves. Tests are the source of
truth for acceptance criteria.

**Scope:** All Playwright e2e tests in `apps/dms-material-e2e/`. Fix Chromium first, then Firefox,
looping until both browsers are fully green.

**Reference:** For clarification about what a specific test is validating, refer to the epic and
story that originally created it, found in
`_bmad-output/planning-artifacts/epics-2026-03-31.md`. Do not use that file as a
source of new work — it is reference material only.

### Story 42.1: Run and Fix Failing Chromium E2E Tests

**Slug:** `42-1-fix-failing-chromium-e2e-tests`

As a developer,
I want all Playwright e2e tests to pass in Chromium,
so that the implementation matches the acceptance criteria defined in previous sprints.

**Acceptance Criteria:**

**Given** the full Playwright e2e suite,
**When** `pnpm run e2e:dms-material:chromium` is executed,
**Then** all tests pass with zero failures.

**Given** a failing test,
**When** the root cause is investigated,
**Then** the fix is applied to the implementation code, not to the test.

**Given** a test whose intent is unclear,
**When** clarification is needed,
**Then** the corresponding epic/story in `_bmad-output/planning-artifacts/epics-2026-03-31.md`
is consulted for context.

**Given** all implementation fixes,
**When** `pnpm all` runs,
**Then** all unit tests continue to pass (no regressions introduced by the fixes).

**Definition of Done:**
- [ ] `pnpm run e2e:dms-material:chromium` exits with 0 failures
- [ ] No test files were modified (only implementation code)
- [ ] `pnpm all` passes

### Story 42.2: Run and Fix Failing Firefox E2E Tests

**Slug:** `42-2-fix-failing-firefox-e2e-tests`

As a developer,
I want all Playwright e2e tests to pass in Firefox,
so that the implementation is browser-agnostic and fully satisfies all acceptance criteria.

**Acceptance Criteria:**

**Given** the full Playwright e2e suite with Chromium tests already passing (Story 42.1 complete),
**When** `pnpm run e2e:dms-material:firefox` is executed,
**Then** all tests pass with zero failures.

**Given** a test that passes in Chromium but fails in Firefox,
**When** the root cause is investigated,
**Then** the fix targets browser-compatibility in the implementation code, not the test.

**Given** all Firefox fixes applied,
**When** `pnpm run e2e:dms-material:chromium` is re-run,
**Then** it still passes (no chromium regressions from the firefox fixes).

**Given** all implementation fixes,
**When** `pnpm all` runs,
**Then** all unit tests continue to pass.

**Definition of Done:**
- [ ] `pnpm run e2e:dms-material:firefox` exits with 0 failures
- [ ] `pnpm run e2e:dms-material:chromium` still exits with 0 failures
- [ ] No test files were modified (only implementation code)
- [ ] `pnpm all` passes
