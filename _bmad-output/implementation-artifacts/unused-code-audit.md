## Classification Criteria

A component is a removal candidate only when ALL five conditions are true:

1. Zero `import` references in any other `*.ts` file (excluding own spec)
2. Not declared in any NgModule
3. Not registered in any router `loadComponent` or `loadChildren`
4. Not referenced in any `*.html` template by selector
5. Not referenced in any `*.stories.ts` file (none exist тАФ auto-satisfied)
