# Epic AO: Implement Fidelity Transaction Import

## Epic Goal

Allow users to import transaction data from Fidelity CSV files.

## Epic Description

Creates import routine to parse Fidelity transaction exports and populate trades/dividends.

## Stories

1. **Story AO.1:** Design Fidelity CSV parser and data mapper
2. **Story AO.2:** Create import service and backend endpoint
3. **Story AO.3:** Build import UI dialog on Global/Universe screen
4. **Story AO.4:** Implement file upload and processing
5. **Story AO.5:** Add validation and error reporting
6. **Story AO.6:** Add unit tests for parser
7. **Story AO.7:** Add unit tests for import service
8. **Story AO.8:** Add e2e tests for import flow

## Dependencies
